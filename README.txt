# avaliacao_php

Trata-se de um exemplo de como realizar uma busca no MySQL utilizando PHP, curl e ajax. O curl está em um arquivo separado por motivo de especificação.

O arquivo database/CriaDatabase.qsl cria a base de dados e gera uma carga com 4 nomes para exemplificar a busca.

A busca é feita pelo nome ou por parte do mesmo.
 Ex: 'Carlos' ou 's'
